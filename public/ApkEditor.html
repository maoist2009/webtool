📖 URL Parameters

Supported URL parameters:

  • seed - Random seed for derivation
  • masterKey - Master signing key
  • signingSeed - Custom signing seed (overrides derived)
  • package - Custom package name
  • packageMode - derived/custom/keep
  • versionCode - Custom version code
  • versionName - Custom version name
  • appName - Custom app name
  • renamePerms - 1/0 to rename permissions

Example: ?masterKey=mykey&packageMode=keep
